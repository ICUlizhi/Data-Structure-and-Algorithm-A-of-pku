## 🔧 使用方法

### 方案1: GitHub Pages (推荐)
1. 进入仓库的 Settings -> Pages
2. 选择 Source 为 "Deploy from a branch"
3. 选择 Branch 为 "main" 和 "/ (root)"
4. 点击 Save
5. 访问 `https://ICUlizhi.github.io/Data-Structure-and-Algorithm-A-of-pku`

### 方案2: 本地构建
```bash
# 安装依赖
npm install

# 构建HTML
npm run build

# 本地预览
npm run serve
```

访问 `http://localhost:8000` 查看结果
